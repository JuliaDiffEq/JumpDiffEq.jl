1 """
2 $(TYPEDEF)
3
4 Highly efficient integrator for pure jump problems that involve only `ConstantRateJump`s,
5 `MassActionJump`s, and/or `VariableRateJump`s *with rate bounds*.
6
7 ## Notes
8 - Only works with `JumpProblem`s defined from `DiscreteProblem`s.
9 - Only works with collections of `ConstantRateJump`s, `MassActionJump`s, and
10 `VariableRateJump`s with rate bounds.
11 - Only supports `DiscreteCallback`s for events, which are checked after every step taken by
12 `SSAStepper`.
13 - Only supports a limited subset of the output controls from the common solver interface,
14 specifically `save_start`, `save_end`, and `saveat`.
15 - As when using jumps with ODEs and SDEs, saving controls for whether to save each time a
16 jump occurs are via the `save_positions` keyword argument to `JumpProblem`. Note that when
17 choosing `SSAStepper` as the timestepper, `save_positions = (true,true)`, `(true,false)`,
18 or `(false,true)` are all equivalent. `SSAStepper` will save only the post-jump state in
19 the solution object in each of these cases. This is because solution objects generated via
20 `SSAStepper` use piecewise-constant interpolation, and can therefore exactly reconstruct
21 the sampled jump process path with knowing just the post-jump state. That is, `sol(t)`
22 for any `0 <= t <= tstop` will give the exact value of the sampled solution path at `t`
23 provided at least one component of `save_positions` is `true`.
24
25 ## Examples
26 SIR model:
27 ```julia
28 using JumpProcesses
29 β = 0.1 / 1000.0; ν = .01;
30 p = (β,ν)
31 rate1(u,p,t) = p[1]*u[1]*u[2] # β*S*I
32 function affect1!(integrator)
33 integrator.u[1] -= 1 # S -> S - 1
34 integrator.u[2] += 1 # I -> I + 1
35 end
36 jump = ConstantRateJump(rate1,affect1!)
37
38 rate2(u,p,t) = p[2]*u[2] # ν*I
39 function affect2!(integrator)
40 integrator.u[2] -= 1 # I -> I - 1
41 integrator.u[3] += 1 # R -> R + 1
42 end
43 jump2 = ConstantRateJump(rate2,affect2!)
44 u₀ = [999,1,0]
45 tspan = (0.0,250.0)
46 prob = DiscreteProblem(u₀, tspan, p)
47 jump_prob = JumpProblem(prob, Direct(), jump, jump2)
48 sol = solve(jump_prob, SSAStepper())
49 ```
50 see the
51 [tutorial](https://docs.sciml.ai/JumpProcesses/stable/tutorials/discrete_stochastic_example/)
52 for details.
53 """

13 randn([rng=default_rng()], [T=Float64], [dims...])
14
15 Generate a normally-distributed random number of type `T`
16 with mean 0 and standard deviation 1.
17 Optionally generate an array of normally-distributed random numbers.
18 The `Base` module currently provides an implementation for the types
19 [`Float16`](@ref), [`Float32`](@ref), and [`Float64`](@ref) (the default), and their
20 [`Complex`](@ref) counterparts. When the type argument is complex, the values are drawn
21 from the circularly symmetric complex normal distribution of variance 1 (corresponding to real and imaginary part having independent normal distribution with mean zero and variance `1/2`).
22
23 See also [`randn!`](@ref) to act in-place.

45 performance for scalar random normal numbers than repeating the code of `_randn`

71 # this unlikely branch is put in a separate function for better efficiency
72 @noinline function randn_unlikely(rng, idx, rabs, x)
73 @inbounds if idx == 0
74 while true
75 xx = -ziggurat_nor_inv_r*log(rand(rng))
76 yy = -log(rand(rng))
77 yy+yy > xx*xx &&
78 return (rabs >> 8) % Bool ? -ziggurat_nor_r-xx : ziggurat_nor_r+xx
79 end
80 elseif (fi[idx] - fi[idx+1])*rand(rng) + fi[idx+1] < exp(-0.5*x*x)
81 return x # return from the triangular area
82 else
83 return randn(rng)
84 end
85 end
